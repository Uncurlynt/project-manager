
# task-manager

Система для управления проектами и задачами. Построен на базе Vue 3 с полной типизацией на TypeScript.

##  Стек технологий
- Framework: Vue 3 

- UI Library: Vuetify

- State Management: Pinia

- Routing: Vue Router 

- HTTP Client: Axios

## Основные возможности

Управление, создание и редактирование задач: детальное модальное окно для управления заголовком, описанием, приоритетом и исполнителем.


Релизованы навигация и фильтрация по проектам и статусам

Поддерживаются светлая и темная темы

## Установка и запуск
Клонирование репозитория:

```sh
git clone https://github.com/Uncurlynt/project-manager.git
```

Установка зависимостей:

```sh
npm install
```

Запуск в режиме разработки:

```sh
npm run dev
```

Для запуска бэкэнда через Docker:

```sh
cd server
make initial-start 
```

Если  хотите запустить напрямую через Go, убедитесь, что у Go установлен и выполните команду:

```sh
cd server
go run сmd/service/main.go
```
